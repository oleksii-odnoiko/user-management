User manager app

## Quickstart

Intsall dependencies:
```sh
  npm run init
  ```

Run app in development mode (http://localhost:4200/):
```sh
  npm run start:dev
   ```

Build app for production:
```sh
  npm run build
   ```

Run app in production (Locally: http://localhost:8080/):
```sh
  npm run start
   ```
